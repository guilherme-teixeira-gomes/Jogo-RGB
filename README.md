# Jogo-RGB

<p> Um jogo no qual o usuário tenta descobrir qual cor RGB é a correta,
  na parte do header irá aparecer um numero rgb, exemplo: (125, 172, 237)
  e na parte debaixo irá aparecer 6 squares com cores diferentes, quando o
  usuário acertar a cor correta, os squares ficarão todos da mesma cor.</p>
  
  <p> Este projeto foi feito atráves de HTML, CSS e Javascript, abaixo irá ter um slide
  mostando um pouco como o jogo funciona.
